Zwin - Responsive Vcard Template

Zwin - Responsive Vcard Template is a clean and simple html template that will help you set up an online Vcard. it is easily usable with any device (Desktop, tablet, mobile phone...), without removing any content!

How To Edit Main Color

If you would like to edit the Main color, you would do the following:

1 - Open this file : css/styles.css

2 - Look for : /*mainColor*/#fdc750

3 - Replace it by your own color.

Change Map Address

If you would like to add you address to the map, you would do the following:

1 - Open this file : js/custom.css

2 - Look for : 

   // Needed variables
    var $map                = $('#map'),
        $tabContactClass    = ('tab-contact'),
        $latitude                = '-37.81759',
        $lontitude                = '144.964557';

3 - Change lantitude and longtitude by your own.

Contact form Configuration! All you need is to configure constants located in file php/config.php

Styles Styles customization and highlights

Heads up! I'm using two CSS files in this theme. The first one is a generic reset file. Many browser interpret the default behavior of html elements differently. By using a general reset CSS file, we can work round this. This file also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.
